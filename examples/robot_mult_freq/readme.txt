- Camera parameters description:

    Parameter	Unit	  Description
01) Ncx		[sel]     Number of sensor elements in camera's x direction
02) Nfx		[pix]     Number of pixels in frame grabber's x direction
03) dx		[mm/sel]  X dimension of camera's sensor element (in mm)
04) dy		[mm/sel]  Y dimension of camera's sensor element (in mm)
05) dpx		[mm/pix]  Effective X dimension of pixel in frame grabber
06) dpy		[mm/pix]  Effective Y dimension of pixel in frame grabber
07) Cx		[pix]     Z axis intercept of camera coordinate system
08) Cy		[pix]     Z axis intercept of camera coordinate system
09) sx		[]        Scale factor to compensate for any error in dpx
10) f		[mm]      Principal distance (focus)
11) kappa1	[1/mm^2]  Distortion coefficient
12) Tx		[mm]	  X Translation
13) Ty		[mm]	  Y Translation
14) Tz		[mm]	  Z Translation
15) Rx		[rad]	  X Rotation
16) Ry		[rad]	  Y Rotation
17) Rz		[rad]     Z Rotation
18) p1		[1/mm]
19) p2		[1/mm]

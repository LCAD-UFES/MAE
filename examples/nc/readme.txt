Neural Computer (NC)

Para utilizar o NC, compile utilizando o comando:
> make

Para programa-lo veja o exemplo em nc.cml. Para rodar o exemplo digite:
> ./nc nc.cml

O comando acima executará a aplicação MAE que implementa o NC e rodará 
o script nc.cml. Este script treina o NC para executar o programa
Contar de 0 a 9 tres vezes e executa 300 "passos" deste programa.

Para saber como programar o NC examine o script nc.cml, o
arquivo de dados DATA/train_data_set.txt, e o documento 
Trabalho de Cognição Visual.doc.

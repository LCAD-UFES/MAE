TUTORIAL DE CONFIGURACAO DO APLICATIVO DE DETECCAO DE FACES (visual_memory) PARA INTEGRA-LO COM OUTRAS REDES NEURAIS DO NC
- PRIMEIRO, DEVEMOS COMPILAR A VERSAO "NO_INTERFACE" DA MAE:
	$ cd $MAEHOME
	$ make -f clean Makefile.no_interface
	$ make -f Makefile.no_interface
- PARA EVITAR ERROS DE REFERENCIA E PERMITIR INTEGRACAO COM OUTROS APLICATIVOS DA MAE QUE NECESSITAM DE INTERFACE,
RENOMEAR UMA COPIA DO DIRETORIO $MAEHOME/lib PARA $MAEHOME/lib.no_interface;
- COMPILAR A VERSAO "NO_INTERFACE" DO EXEMPLO FACE_RECOG_PLANAR: (ISSO VAI GERAR A BIBLIOTECA ESTATICA libface_recog.a)
	$ cd $MAEHOME/examples/face_recog/planar
	$ make -f clean Makefile.no_interface
	$ make -f Makefile.no_interface
- COMPILAR O APLICATIVO DE DETECCAO DE FACES:
	- INSTALAR A BIBLIOTECA OpenCV VERSAO 2.1;
	- NO ARQUIVO "face_detect_api.cpp" SETAR A VARIAVEL basePath PARA SEU DIRETORIO $MAEHOME;
	$ cd $MAEHOME/examples/nc/visual_memory
	$ make
	$ ./face_detect_app
- SE TUDO DER CERTO, O APLICATIVO VAI EXECUTAR E IMPRIMIR NA TELA O ID E NOME DA PESSOA RECONHECIDA (ID=1, Name: vitor);
- OS PASSOS ANTERIORES TAMBEM VAO GERAR A BIBLIOTECA ESTATICA libnc_visual_memory.a;

PARA FACILITAR, FOI CRIADO UM EXEMPLO DE INVOCACAO DO "visual_memory" COMO LIB ESTATICA.
	$ cd $MAEHOME/examples/nc/visual_memory_example
	$ make
	$ ./VisualMemoryExample
